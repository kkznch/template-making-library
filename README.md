Template making library
=======================

## Getting Started
### Prerequisites
- pnpm
  - https://pnpm.io/ja/

### Installation
1. Create repository from this template
2. Clone the repository
    ```sh
    $ git clone git@github.com:YOUR_ACCOUNT/YOUR_REPO
    $ cd YOUR_REPO
    ```
3. Run package install
    ```sh
    $ pnpm install
    ```

## Usage
Write your code, then run below command.

```sh
$ pnpm build
```
